# 📂 Atividade: Algoritmos, Fluxogramas e Pseudocódigos

Este repositório contém a resolução da atividade prática de lógica de programação. O desafio consistia em elaborar algoritmos completos para diferentes problemas, utilizando duas formas de representação: **Fluxogramas** (visual) e **Pseudocódigo** (estruturado).

---

## 🎯 Objetivos do Projeto

* Praticar a análise de problemas e estruturação de soluções lógicas.
* Desenvolver fluxogramas para compreensão do fluxo de dados.
* Implementar pseudocódigos.

---

## 💻 Conteúdo da Entrega

Para cada item da atividade, você encontrará:
1.  **Fluxograma:** Diagrama detalhando a lógica do início ao fim.
2.  **Pseudocódigo:** Algoritmo escrito de forma textual e estruturada.
